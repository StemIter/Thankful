# 照片文件说明

请在此文件夹放置用于展示的照片：

## 照片分类
- `team/` - 团队合影和交流照片
- `work/` - 工作瞬间和项目研讨照片
- `education/` - 劳动STEM教育相关照片
- `research/` - 课题研究和学术讨论照片
- `activities/` - 文体活动和文艺表演照片

## 照片尺寸建议
- 横向照片：建议 1600x900 像素或类似比例
- 纵向照片：建议 900x1200 像素或类似比例
- 格式：JPG、PNG、WebP

## 使用方式
照片URL可直接在组件中引用，例如：
```typescript
const photos = [
  '/images/team/group-photo-1.jpg',
  '/images/education/class-discussion.jpg',
  // ...
];
```
